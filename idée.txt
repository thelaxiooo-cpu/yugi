  (chaque booster contient 3 cartes)


▌ ÉVÉNEMENT COUPE DU MONDE
──────────────────────────────────────────────────
  → 1h avant le match : bot ping @everyone avec
    "Match dans 1h : Équipe A vs Équipe B"
  → Joueurs soumettent /prono 2-1 (modifiable)
  → Les pronostics ferment 5 min avant le match
    (le bot envoie un message de fermeture)
  → Résultats entrés par un admin (/admin result)
  → Récompenses automatiques :
      Score exact     → +3 boosters
      Bon vainqueur   → +1 booster


▌ LISTE DES COMMANDES
──────────────────────────────────────────────────

  /in           S'inscrire comme joueur (obligatoire)
  /help         Aide de toutes les commandes
  /dc           Gagner 100–1000 DC (1 fois par jour)
  /me           Voir son profil (DC, level, cartes, boosters)
  /shop         Acheter des boosters (boutons interactifs)
  /open         Ouvrir un booster (affiche les 5 cartes + images)
  /deck         Voir sa collection triée par rareté et par nom
  /lvl          Monter de niveau
  /drops        Voir les taux de drop par niveau
  /prono 2-1    Soumettre un pronostic pour le match en cours

  — ADMIN UNIQUEMENT —
  /admin give-boosters  Donner des boosters à un joueur
  /admin give-coins     Donner des DC à un joueur
  /admin add-match      Ajouter un match (équipe1, équipe2, datetime)
  /admin result         Entrer le score + distribuer les récompenses
  /admin matches        Lister les matchs à venir
  /admin simulate-match Tester la notification de match
